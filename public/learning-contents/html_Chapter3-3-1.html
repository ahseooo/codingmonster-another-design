🖼️ 이미지 넣기 (Image)

글자만 있으면 심심하잖아? 🎨
<img> 태그로 멋진 사진을 넣어보자! (닫는 태그 없음!)

웹페이지에 이미지를 넣을 때는 <img> 태그를 사용해요.
어떤 이미지를 보여줄지 주소(src)를 꼭 적어줘야 한답니다.

1. 이미지 태그 구조

<img>는 닫는 태그 없이 혼자 쓰이는 빈 태그(Empty Tag)예요.

이미지 태그 해부하기 🔍
<img
src = "cat.jpg" 이미지 주소 (필수!)
alt = "고양이" 설명 (안 보일 때 나옴)
>
HTML Code 
<img src="cat.jpg" alt="귀여운 고양이">
Browser Result
(실제로는 고양이 사진이 나옵니다)

2. 이미지 주소 (src) 종류

src (Source) 속성에 파일의 위치를 정확하게 적어야 이미지가 나와요.

같은 폴더에 있을 때 src="photo.jpg"
하위 폴더(img)에 있을 때 src="img/photo.jpg"
인터넷 주소일 때 src="https://site.com/img.jpg"
alt 속성은 왜 쓰나요?
  • ❌
    이미지가 깨졌을 때
    인터넷이 느리거나 파일명이 틀리면 대신 글씨가 나와요.
  • 👂
    시각 장애인을 위해
    화면 낭독기(Screen Reader)가 이 설명을 읽어줘요. (매우 중요!)
