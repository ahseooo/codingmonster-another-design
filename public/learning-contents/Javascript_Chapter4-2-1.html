🏃‍♂️ 객체 메서드 (Method)

객체는 정보만 담는 게 아니야! 🙅‍♂️
스스로 행동할 수 있는 능력(함수)도 가질 수 있어.

메서드(Method)는 객체 안에 들어있는 함수를 말해요.
강아지가 짖거나, 자동차가 달리는 것처럼 객체의 행동을 정의할 수 있어요.

1. 메서드 만들기 (행동 추가)

강아지 객체에 bark(짖기)라는 행동을 넣어볼까요?

JavaScript 
let dog = {
  name: "초코",
  bark: function() {
    console.log("멍멍! 🐶");
  }
};

dog.bark(); // 실행!
Console Result
🐕 💨 "멍멍!"
🎭
비유: name은 강아지의 이름표이고,
bark는 강아지가 할 수 있는 개인기예요!

2. 자기 자신 가리키기 (this)

메서드 안에서 내 정보(속성)를 쓰고 싶을 땐 this를 써요.

JavaScript 
let dog = {
  name: "초코",
  sayHello: function() {
    console.log("안녕! 난 " + this.name);
  }
};

dog.sayHello();
Result
안녕! 난 초코
🐶
👈
여기서 this는 "나 자신(dog 객체)"을 가리켜요.
this.name은 곧 dog.name과 같아요!

3. 여러 가지 행동 넣기

하나의 객체에 여러 개의 메서드를 넣을 수 있어요.

dog.bark()
🐶 멍멍!
dog.sleep()
💤 쿨쿨...
dog.run()
🏃‍♂️ 다다다!
핵심 포인트
  • ⚙️
    메서드 (Method)
    객체 속에 들어있는 함수. (객체의 행동)
  • 👆
    this
    메서드가 속해 있는 객체 자신을 가리키는 키워드.
