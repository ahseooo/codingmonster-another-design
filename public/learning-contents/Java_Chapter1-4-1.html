💻 이클립스(Eclipse) 설치하기

IDE는 Integrated Development Environment(통합 개발 환경)의 약자야.

메모장으로도 코딩할 수 있지만, IDE를 쓰면 훨씬 빠르고 편하게 개발할 수 있어.
자바에서는 이클립스(Eclipse)를 가장 많이 사용해!

📌 IDE가 해주는 것들

📝
코드 작성 도움
자동 완성: Sysout → System.out.println() 자동으로 완성!
코드 하이라이트: 필터링, 색상 표시로 코드 흐름을 쉽게 이해!
자동 정렬: 보기 쉽게 코드 자동 정렬!
🔍
오류 찾기
오류 표시: 빨간 줄로 오류를 알려줘!
오류 위치로 자동 이동!
오류 원인: 어떤 문제가 있는지 설명까지 제공!
▶️
실행과 디버깅
버튼 클릭한 번으로 실행!
디버깅: 코드가 한 줄씩 어떻게 실행되는지 확인 가능!
에러 위치 즉시 확인 & 변수 값 확인!

📌 다양한 IDE 종류

추천
🌙
Eclipse
자바(Java) 개발에 가장 널리 쓰이는 IDE
큰 프로젝트도 다루는 데 특화!
🖥️
Visual Studio
마이크로소프트에서 만든 IDE
C#이나 .NET 개발에서 많이 사용.
💙
Visual Studio Code
가볍고 빠른 IDE
다양한 언어와 플러그인을 지원해서 누구나 쉽게 쓸 수 있어.
🧠
IntelliJ IDEA
자동 완성과 기능이 매우 강력한 IDE
자바, 코틀린(Kotlin) 개발에 특화되어 있어.
🐍
PyCharm
파이썬(Python) 전용 IDE
파이썬 개발자들이 가장 많이 사용하는 도구야.
