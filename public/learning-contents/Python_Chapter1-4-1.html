💻 PyCharm이 뭐야?

PyCharm은 파이썬 전용 IDE야.
코드 작성, 실행, 디버깅을 한 프로그램 안에서 편하게 할 수 있도록 도와주는 도구야!

📌 메모장 대신 PyCharm사용하면 좋은 이유

📝
문법 하이라이트 표시
코드 문법의 하이라이트 기능으로 코드의 내용을 구분하기 편해.
🖥️
자동 완성 기능
자동 완성기능으로 빠르게 입력할 수 있어.
🎯
오류 표시
오류를 빨간 줄로 표시해서 알려줘.
📁
파일 관리
여러 파일을 프로젝트로 깔끔하게 관리할 수 있어.
