🗂️ JSON과 데이터 구조

컴퓨터끼리 대화할 때 쓰는 공통 언어가 있다고? 🗣️
정보를 깔끔하게 포장해서 주고받는 JSON을 알아보자!

JSON은 데이터를 저장하거나 주고받을 때 사용하는 텍스트 형식이에요.
사람도 읽기 쉽고 컴퓨터도 이해하기 쉬워서 전 세계적으로 쓰인답니다.

1. JSON의 생김새

이름(Key)과 값(Value)이 쌍으로 이루어져 있어요.

JSON Format
{
  "name": "철수",
  "age": 10,
  "likes": ["게임", "축구"],
  "isStudent": true
}
Information
👦
  • 이름: 철수
  • 나이: 10세
  • 취미: 게임, 축구
  • 학생: 네 (True)

2. 데이터 구조 (Data Structure)

데이터를 어떤 모양으로 정리해서 보관할지 결정하는 방법이에요.

🔢

배열 (Array)

순서대로 줄 세우기

예: 번호순 출석부
🏷️

객체 (Object)

이름표 붙여서 담기

예: 이름표 붙은 사물함
📚

리스트 (List)

목록 만들기

예: 할 일 목록(To-Do)

3. JSON vs 데이터 구조

둘은 짝꿍이지만 역할이 달라요!

📄

JSON

데이터를 적는 형식(Format)

규칙
🤝
🗂️

데이터 구조

데이터를 정리하는 방법(Method)

정리 정돈
JSON 안에서도 배열 []과 객체 {} 구조를
자유롭게 섞어서 사용할 수 있어요!
핵심 요약
  • JSON: 데이터를 텍스트로 표현하는 규칙 (이름표: 값).
  • 배열: 순서가 중요한 데이터 (1번, 2번, 3번...).
  • 객체: 이름(Key)으로 데이터를 찾는 구조.
