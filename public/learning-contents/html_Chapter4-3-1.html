📝 폼(Form) 만들기

회원가입이나 로그인 창은 어떻게 만들까? 🤔
사용자에게 정보를 입력받는 폼(Form)을 만들어보자!

폼은 사용자가 입력한 정보를 서버로 보내기 위한 양식이에요.
다양한 입력 도구들을 모아서 하나의 세트로 만들죠.

1. 폼의 기본 구조 (<form>)

<form> 태그는 입력 요소들을 감싸는 가방 역할을 해요.

HTML Code 
<form action="/login" method="POST">
  <!-- 여기에 입력 칸들을 넣어요 -->
</form>
속성 설명
  • action: 입력 정보를 보낼 목적지(주소)
  • method: 정보를 보내는 방법(GET/POST)

2. 다양한 입력 태그들

상황에 맞는 도구를 골라 써보세요.

<input type="text">
<input type="password">
<textarea>
<select>
<input type="checkbox">
