웹 페이지의 뼈대,
문서 구조 익히기

제목, 문단, 그리고 다양한 태그들을 조합하여
튼튼하고 짜임새 있는 웹 페이지를 만들어보세요!

학습 목차

  • HTML 문서의 기본 구조 (html, head, body)
  • 제목(Heading)과 문단(Paragraph)
  • 줄바꿈, 수평선, 그리고 주석
  • 특수문자, 들여쓰기, 텍스트 정렬

학습 목표

  • HTML 문서의 필수 태그와 구조를 이해한다.
  • 텍스트를 제목과 본문으로 구조화할 수 있다.
  • 코드의 가독성을 높이는 방법(주석, 들여쓰기)을 익힌다.
  • 특수문자와 정렬 속성을 활용해 문서를 꾸밀 수 있다.
