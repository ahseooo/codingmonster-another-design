🛠️ 자바스크립트는 어떻게 만드나요?

자바스크립트는 혼자서는 못 놀아! 👯‍♂️
항상 HTML 친구랑 같이 다닌단다. 어떻게 연결하는지 볼까?

자바스크립트는 HTML 문서 안에 직접 적을 수도 있고,
따로 파일을 만들어서 불러올 수도 있어요.

1. HTML 안에 직접 쓰기 (<script>)

가장 쉬운 방법이에요. HTML 파일 안에 <script> 태그를 만들고 그 안에 코드를 적어요.

HTML Code 
<body>
  <h1>안녕하세요!</h1>

  <!-- 자바스크립트는 주로 body 끝에 넣어요 -->
  <script>
    alert('환영합니다!');
  </script>
</body>
Result

2. 따로 파일 만들기 (외부 스크립트)

코드가 길어지면 .js 파일로 따로 저장해서 불러오는 게 좋아요. (추천! 👍)

JS
script.js
alert('안녕!');
연결 (src)
HTML
index.html
<script src="script.js"></script>

3. 사용자와 놀기 (이벤트)

클릭하거나 마우스를 올렸을 때 반응하도록 만들 수 있어요.

👇 버튼을 클릭해보세요!

👇 마우스를 올려보세요!

마우스 오버
