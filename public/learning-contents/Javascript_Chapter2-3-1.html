➕ 연산자 (Operator)

컴퓨터는 계산기야! 🧮
더하고 빼는 산술 연산과, 누가 더 큰지 따지는 비교 연산을 배워보자.

연산자는 프로그래밍의 계산 버튼과 같아요.
숫자를 계산하거나(산술), 두 값을 비교해서 참/거짓을 판단할 때(비교) 사용합니다.

1. 산술 연산자 (Arithmetic)

사칙연산과 비슷해요. 나머지 연산(%)은 코딩에서 정말 자주 쓰이니 꼭 기억하세요!

+
더하기
2 + 3 = 5
-
빼기
5 - 2 = 3
*
곱하기
3 * 4 = 12
/
나누기
10 / 2 = 5
%
나머지
10 % 3 = 1
JavaScript 
let a = 10;
let b = 3;
console.log(a % b); // 10 나누기 3의 나머지
Result

1

2. 비교 연산자 (Comparison)

두 값을 비교해서 참(true) 또는 거짓(false)을 알려줘요.

>
크다
5 > 3
true
<
작다
2 < 8
true
>=
크거나 같다
3 >= 3
true
==
같다 (값만)
5 == 5
true
===
완전 같다 (타입까지)
5 === "5"
false
!=
다르다
5 != 3
true
⚖️
비유: 양팔 저울에 물건을 올리는 것과 같아요.
"왼쪽이 더 무거운가?"라고 물어보면 저울이 YES(true)나 NO(false)로 답해주는 거죠!
