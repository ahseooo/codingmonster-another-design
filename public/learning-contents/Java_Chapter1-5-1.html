자주 발생하는 에러와 해결방법

프로그래밍을 하다 보면 에러는 자연스러운 일이니 너무 걱정하지 마!
대부분의 에러는 간단하게 해결할 수 있어. 가장 자주 발생하는 에러와 해결 방법을 알아볼까?

📁 파일 이름 오류

클래스 이름과 파일 이름이 다르면 오류가 발생해!

잘못된 예
/* Hello.java 파일 */
public class Test { }
올바른 예
/* Hello.java 파일 */
public class Hello { }
파일 이름과 클래스 이름은 반드시 동일해야 해!
대소문자도 정확하게 작성해야 해.
🖥️ 컴파일 오류

세미콜론(;)이 없으면 컴파일 오류가 발생해!

잘못된 예
System.out.println("hello")
올바른 예
System.out.println("hello");
문장의 마지막에는 반드시 세미콜론을 붙여야 해!
✏️ 대소문자 구분

자바는 대소문자를 구분해!

잘못된 예
system.out.println();
올바른 예
System.out.println();
System의 'S'는 반드시 대문자여야 해!
자바는 철저하게 대소문자를 구분해.
⚙️ 경로 설정 오류

JAVA 또는 javac 명령어가 동작하지 않는다면 환경 변수 설정 문제일 수 있어!

오류 메시지
'javac'은(는) 내부 또는 외부 명령,
실행할 수 있는 프로그램 또는
배치 파일이 아닙니다.
JAVA_HOME과 Path 설정을 다시 확인해봐!

✔ JAVA_HOME → JDK 설치 경로 지정
✔ Path → %JAVA_HOME%\bin 추가

에러 해결 꿀팁!

  • 에러 메시지를 무시하면 안돼! 가장 중요한 힌트야.
  • 에러가 표시된 줄 번호부터 확인하기.
  • Ctrl + S 로 자주 저장하기.
  • 문제가 해결되지 않으면 구글 검색으로 빠르게 해결!
