✅ HTML의 기본 뼈대

안녕! 나는 HTML 몬스터야.
사람에게 뼈가 있듯이, 웹 페이지도 기본 구조가 있어!

HTML 문서는 크게 설정 정보가 들어가는 머리(Head)와 내용이 들어가는 몸통(Body)으로 나뉘어져 있어.

기본 구조 예시 

<!DOCTYPE html>
<html>
  <head>
    <title>내 웹페이지 제목</title>
  </head>
  <body>
    <h1>안녕하세요!</h1>
    <p>이건 내 첫 번째 웹페이지에요!</p>
  </body>
</html>
