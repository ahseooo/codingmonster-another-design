⚡ 브라우저 이벤트 (Browser Events)

클릭! 🖱️ 키보드 타닥! ⌨️
사용자가 무언가 할 때, "짠!" 하고 반응하게 만들어볼까?

이벤트(Event)는 웹페이지에서 일어나는 모든 사건을 말해요.
HTML 태그에 "이 일이 일어나면 이거 해!"라고 붙이는 마법 스위치예요. 🪄

1. 자주 쓰는 이벤트 종류

🖱️
onClick
클릭하면 동작
🔄
onChange
값이 바뀌면 동작
⌨️
onInput
입력할 때마다 동작
👆
onMouseOver
마우스 올리면 동작
🎹
onKeyDown
키 누르면 동작

2. onClick (클릭했을 때)

버튼이나 이미지를 클릭했을 때 실행돼요.

<button onclick="alert('눌렀다!')">
  눌러봐!
</button>
Demo
🔔
비유: 초인종을 눌렀더니 "띠링!" 하고 벨이 울리는 것과 같아요.

3. onChange (값이 바뀔 때)

선택창(Select)이나 체크박스의 값이 변경되면 실행돼요.

<select onchange="alert('바뀌었어!')">
  <option>사과</option>
  <option>바나나</option>
</select>
Demo
🥤
비유: 자판기에서 음료를 선택하는 순간 불이 들어오는 것과 같아요.

4. onMouseOver (마우스 올렸을 때)

요소 위에 마우스 커서가 올라가면 실행돼요.

<div onmouseover="...">
  상자
</div>
Demo
마우스를 올려보세요
💡
비유: 센서등 근처만 가도 자동으로 불이 켜지는 것과 같아요.
이벤트 정리
이벤트 발생 시점 비유
onClick 클릭할 때 🔔 초인종 누르기
onChange 값이 바뀔 때 🎛 채널 돌리기
onInput 입력할 때 ✍️ 실시간 메모
onMouseOver 마우스 올릴 때 💡 센서등 켜짐
