🎭 조건부 렌더링 (Conditional Rendering)

조건에 따라서 보여줄지 말지 결정하는 마법! ✨
"숙제를 다 했으면 스티커를 보여줘!" 같은 거야.

조건부 렌더링은 특정 조건이 참(true)일 때만 화면에 무언가를 표시하거나,
상황에 따라 다른 내용을 보여주는 기술이에요.

1. 조건에 따라 다르게 보여주기

if문을 사용해서 상황을 판단해요.

JavaScript Logic
let isStudent = true;

if (isStudent) {
  console.log("학생이에요! 🎒");
} else {
  console.log("학생이 아니에요! 👔");
}
Result
학생이에요! 🎒
⚖️
비유: 놀이공원에서 "키가 120cm 넘으면 탈 수 있어요!" 라고 검사하는 것과 같아요.
조건이 맞으면 타고, 아니면 못 타죠!

2. 화면(HTML)을 껐다 켰다 하기

자바스크립트로 HTML 요소를 보이게 하거나 숨길 수 있어요.

JavaScript
function toggleMessage() {
  let msg = document.querySelector("#msg");
  
  if (msg.style.display === "none") {
    msg.style.display = "block"; // 보이기
  } else {
    msg.style.display = "none";  // 숨기기
  }
}
