📊 표 만들기 (Table)

성적표나 시간표처럼 정보를 깔끔하게 정리하고 싶다고?
표(Table) 태그를 사용하면 행과 열로 정리할 수 있어!

표는 여러 태그들이 모여서 만들어집니다.
큰 상자(table) 안에 줄(tr)을 만들고, 그 안에 칸(td)을 채워 넣는 방식이에요.

1. 표를 만드는 기본 태그

표는 <table>, <tr>, <td> 세 가지가 기본 세트예요.

<table>
표 전체를 감싸는 틀
<tr>
Table Row (가로 줄)
<td>
Table Data (데이터 칸)
<th>
Table Header (제목 칸)

2. 표 만들기 실습

학생 성적표를 만들어봅시다. 제목 칸은 <th>를 써서 굵게 표시해요.

HTML Code 
<table>
  <tr>
    <th>이름</th>
    <th>수학</th>
  </tr>
  <tr>
    <td>철수</td>
    <td>90점</td>
  </tr>
  <tr>
    <td>영희</td>
    <td>95점</td>
  </tr>
</table>
Browser Result
이름 수학
철수 90점
영희 95점

3. 표의 구조 나누기 (선택 사항)

표가 복잡해지면 머리(Head), 몸통(Body), 바닥(Foot)으로 구역을 나눌 수 있어요.

표의 3단 구조
  • 🧠
    <thead>
    표의 머리글 (제목 행)을 감싸요.
  • 🦴
    <tbody>
    표의 본문 데이터들을 감싸요.
  • 🦶
    <tfoot>
    표의 바닥글 (합계 등)을 감싸요.
