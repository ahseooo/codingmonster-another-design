📦 객체 (Object) 만들기

이름, 나이, 사는 곳... 📝
관련된 정보를 한 상자(객체)에 깔끔하게 담아보자!

객체는 여러 개의 데이터(속성)를 키: 값 형태로 묶어서 관리하는 자료형이에요.
마치 사람의 프로필 카드와 같죠!

1. 객체 만들기 ({ })

중괄호 { }를 열고 정보를 담아요.

JavaScript 
let dog = {
  name: "초코",
  age: 3,
  color: "갈색"
};
Object Info
🐕

이름: 초코

나이: 3살

색상: 갈색

🗂️
비유: 변수가 하나씩 따로 노는 낱장 종이라면,
객체는 정보를 모아놓은 파일철이에요!

2. 정보 꺼내기 (. 또는 [])

점(.)을 찍거나 대괄호([])를 써서 원하는 정보를 가져와요.

JavaScript 
console.log(dog.name);
console.log(dog["age"]);
Console Result
  • "초코"
  • 3

3. 정보 추가하고 수정하기

새로운 정보를 넣거나, 기존 정보를 바꿀 수 있어요.

JavaScript 
dog.age = 4; // 나이 수정
dog.sound = "멍멍!"; // 소리 추가

console.log(dog);
Updated Object
🐕

이름: 초코

나이: 4살

색상: 갈색

소리: 멍멍!

객체 사용법
동작 방법 예시
만들기 { 키: 값 } let a = { x: 10 }
꺼내기 . 또는 [""] a.x 또는 a["x"]
수정/추가 객체.키 = 값 a.x = 20
