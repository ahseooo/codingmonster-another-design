🐞 자바스크립트 에러 해결하기

빨간색 에러 글씨가 떴다고? 😱 당황하지 마!
에러 메시지만 잘 읽어도 금방 고칠 수 있어.

개발자들도 매일 에러를 만나요.
자주 발생하는 에러 6가지를 알고 있으면 훨씬 빨리 문제를 해결할 수 있어요!

1. 문법이나 변수가 틀렸을 때

SyntaxError: Unexpected token

👉 문법이 어딘가 이상해요! (괄호, 따옴표 등)

if (x > 10 {
✅ 해결: 닫는 괄호 )가 빠졌네요!
ReferenceError: is not defined

👉 변수를 안 만들고 썼어요!

console.log(name);
✅ 해결: let name = "지우"; 처럼 먼저 선언하세요.
TypeError: Cannot read property...

👉 없는 것(undefined)에서 무언가를 찾으려 했어요.

let user; console.log(user.name);
✅ 해결: 변수에 값이 잘 들어있는지 확인하세요.

2. 계산이나 값이 이상할 때

TypeError: ... is not a function

👉 함수가 아닌데 함수처럼 실행(...)했어요.

let num = 10; num();
✅ 해결: 변수 이름이나 타입을 확인하세요.
NaN (Not a Number)

👉 숫자가 아닌 걸로 계산했어요.

"안녕" * 5
✅ 해결: 계산하려는 값이 진짜 숫자인지 확인하세요.
Infinite Loop (무한 루프)

👉 반복문이 안 멈추고 계속 돌아요! (브라우저 멈춤)

✅ 해결: while이나 for문의 종료 조건이 맞는지 꼭 확인!
🕵️‍♂️
꿀팁: console.log()를 활용하세요!
코드가 어디까지 실행되는지, 변수 값이 무엇인지 궁금할 땐
console.log(변수명);을 적어서 확인해보는 게 최고예요!
