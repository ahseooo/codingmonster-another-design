🖱️ 요소 콕 집어서 바꾸기 (선택과 조작)

자바스크립트야, 저기 있는 빨간 버튼 좀 눌러줄래? 👉🔴
원하는 요소를 찾고(선택), 내 마음대로 바꾸는(조작) 방법을 배워보자!

웹페이지를 살아 움직이게 하려면, 먼저 바꿀 대상을 정확히 선택해야 해요.
그 다음 글자를 바꾸거나, 색을 칠하거나, 숨길 수 있죠.

1. 요소 선택하기 (Finding Elements)

HTML 태그에 이름표(ID, Class)를 붙여서 찾을 수 있어요.

#ID

getElementById()

주민등록번호처럼 딱 하나만 있는 요소를 찾을 때 써요.

document.getElementById("title")
CSS 선택자

querySelector()

CSS처럼 #id나 .class로 자유롭게 찾을 수 있어요. (추천! 👍)

document.querySelector(".box")

2. 요소 바꾸기 (Manipulating)

찾은 요소를 마음대로 변신시켜 볼까요?

✏️
textContent

글자 내용을 바꿔요.

el.textContent = "바뀜!";
🎨
style

색상, 크기 등 디자인을 바꿔요.

el.style.color = "red";
👻
display

보이거나 숨길 수 있어요.

el.style.display = "none";

3. 실전 연습: 버튼 누르면 변신! 🦸‍♂️

JavaScript 
let text = document.querySelector("#msg");
let btn = document.querySelector("#btn");

btn.addEventListener("click", function() {
  text.textContent = "변신 완료! ✨";
  text.style.color = "purple";
  text.style.fontSize = "24px";
});
