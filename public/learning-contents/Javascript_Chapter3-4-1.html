🚂 배열 조작하기 (push, pop, forEach)

기차에 화물을 싣거나 내리는 것처럼,
배열에 데이터를 추가하고 삭제하는 방법을 알아보자! 📦

이미 만들어진 배열에 새로운 값을 넣거나(push),
마지막 값을 빼거나(pop), 하나씩 꺼내볼 수(forEach) 있어요.

1. 뒤에 추가하기 (push)

기차의 맨 뒤에 새로운 칸을 연결해요.

JavaScript 
let train = ["A", "B"];
train.push("C");
console.log(train);
Train Simulation
🚂
A
B
C

2. 마지막 빼기 (pop)

맨 뒤에 있는 칸을 떼어내요.

JavaScript 
let train = ["A", "B", "C"];
train.pop();
console.log(train);
Train Simulation
🚂
A
B
C

3. 하나씩 꺼내기 (forEach)

배열에 있는 모든 데이터를 하나씩 차례대로 확인해요.

JavaScript 
let fruits = ["사과", "포도"];

fruits.forEach(function(item) {
  console.log(item + " 맛있다!");
});
Console Result
  • 🍎 사과 맛있다!
  • 🍇 포도 맛있다!
🔄
비유: 회전초밥 레일에서 접시를 하나씩 집어서 먹는 것과 같아요! 🍣
배열 메서드 정리
메서드 역할 방향
push() 데이터 추가 맨 뒤 ➕
pop() 데이터 제거 맨 뒤 ➖
forEach() 반복 실행 처음부터 끝까지 🔄
