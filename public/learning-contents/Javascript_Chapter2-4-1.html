💬 조건문 (if, else, switch)

컴퓨터에게 생각하는 법을 알려주자! 🧠
"비가 오면 우산을 쓰고, 안 오면 그냥 가!" 처럼 상황에 따라 다르게 행동하게 할 수 있어.

조건문은 주어진 조건이 참(true)인지 거짓(false)인지에 따라
서로 다른 코드를 실행하는 제어문이에요.

1. 만약에... (if)

조건이 참(True)일 때만 실행해요.

JavaScript 
let isRainy = true;

if (isRainy) {
  console.log("우산을 써요 ☔");
}
Result
우산을 써요 ☔

2. 그게 아니면... (if - else)

조건이 참이면 A를, 거짓이면 B를 실행해요. (양자택일)

날씨가 맑은가?
YES (True)
🌞 산책 가기
NO (False)
🏠 집에 있기
if (isSunny) { console.log("산책 가요 🌞"); } else { console.log("집에 있어요 🏠"); }

3. 여러 조건 따지기 (else if)

조건이 여러 개일 때 순서대로 물어봐요.

JavaScript 
let score = 85;

if (score >= 90) {
  console.log("A학점 🎉");
} else if (score >= 80) {
  console.log("B학점 👍");
} else {
  console.log("C학점 💪");
}
Result
B학점 👍

4. 딱 떨어지는 값 고르기 (switch)

조건이 많고, 값이 명확할 때(예: 메뉴 선택) 사용하면 깔끔해요.

선택: 사과
case "사과": console.log("빨간 사과"); break;
조건문 핵심 정리
  • ☔
    if
    조건이 맞을 때만 실행
  • ⚖️
    else
    조건이 틀렸을 때 실행 (그 외 나머지)
  • 🚦
    switch
    값이 딱 떨어지는 경우 (메뉴 선택 등)
