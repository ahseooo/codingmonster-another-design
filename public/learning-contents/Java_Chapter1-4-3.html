🖥️ 이클립스 화면 구성

이클립스 화면이 어떻게 구성되어 있는지 알아보자!
🗂️ 각 영역 자세히 알아보기

📁 Package Explorer
(왼쪽)

  • 프로젝트와 파일 목록
  • 폴더 구조 확인
  • 파일 클릭으로 열기
  • 새 프로젝트/파일 생성

✏️ Editor Area
(중앙)

  • 코드 작성 영역
  • 자동 완성 기능
  • 문법 색상 표시
  • 오류 빨간 줄 표시

🖥️ Console
(하단)

  • 실행 결과 표시
  • 오류 메시지 확인
  • System.out.println() 출력
  • 프로그램 실행 상태 확인
📝 알아두면 좋아요!

화면 영역 변경하기:
각 영역의 경계선을 드래그하여 크기를 조절할 수 있어!
화면이 사라졌다면 메뉴의 Window → Show View에서 다시 불러올 수 있어!

처음으로 되돌리기:
Window → Perspective → Reset Perspective 를 선택하면 기본 화면 구성으로 돌아가!
