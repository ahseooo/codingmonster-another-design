♿ 웹 접근성 (Web Accessibility)

모두가 함께 쓰는 인터넷! 🌍
장애가 있거나 나이가 많아도 불편함 없이 웹을 사용할 수 있어야 해.

시각장애, 청각장애, 고령자 등 모든 사용자가 웹사이트를 쉽게 이용할 수 있도록 만드는 것이 웹 접근성입니다.

시각
청각
신체
인지

1. 이미지 설명하기 (alt)

시각장애인은 화면 낭독기(스크린 리더)로 웹을 이용해요. 이미지가 무엇인지 글로 알려줘야 합니다.

HTML Code 
<img src="cat.jpg" alt="검은 고양이가 앉아 있는 사진">
Screen Reader
🗣️
"검은 고양이가 앉아 있는 사진, 이미지"

2. 폼(Form)에 이름표 붙이기 (label)

입력창만 있으면 무엇을 입력해야 할지 몰라요. label로 짝을 지어주세요.

HTML Code 
<label for="username">이름:</label>
<input type="text" id="username">
Connection
for="A" id="A"

서로 연결되어 있음을 알려줌!

3. 논리적인 순서 (Heading)

h1 다음에 h2가 와야 해요. 갑자기 h4로 건너뛰면 혼란스러워요.

H1 (페이지 제목)

H2 (중요 주제)

H3 (세부 내용)
