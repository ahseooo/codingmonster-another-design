🔗 하이퍼링크 (Hyperlink)

클릭하면 다른 페이지로 슝~ 이동하는 마법! ✨
앵커(Anchor)의 약자인 <a> 태그를 써보자.

웹(Web)이 거미줄처럼 연결될 수 있는 건 바로 링크 덕분이에요.
다른 사이트로 가거나, 내가 만든 다른 파일로 이동해봅시다.

1. 기본 사용법 (href)

href 속성에 이동할 주소를 적어주세요. (href = Hypertext Reference)

링크 태그 해부하기 🔍
<a
href = "https://naver.com" 이동할 주소
> 네이버로 가기 </a>
HTML Code 
<a href="https://www.naver.com">
  네이버로 가기
</a>
Browser Result

아래 링크를 눌러보세요:

네이버로 가기

2. 새 탭에서 열기 (target)

target="_blank"를 추가하면 현재 페이지는 그대로 두고 새 창을 열어요.

HTML Code 
<a href="https://google.com" target="_blank">
  구글 (새 탭)
</a>
Browser Result
구글 (새 탭)
창이 어떻게 열리나요?
  • 📄
    기본값 (속성 없음)
    현재 보고 있는 창이 바뀌어요. (뒤로가기 눌러야 함)
  • 📑
    target="_blank"
    탭이 하나 더 생겨요. (원래 페이지 유지)

3. 내 파일끼리 이동하기

같은 폴더에 있는 다른 HTML 파일로 이동할 때는 파일 이름만 적으면 돼요.

📂 프로젝트 폴더 구조

내_웹사이트
index.html 현재 위치
about.html 이동할 곳
⬇️ 코드는 이렇게 작성해요 ⬇️
<a href="about.html">소개 페이지</a>
