🔢 자료형 (Data Type)

상자(변수) 안에 무엇을 담을 수 있을까? 🎁
숫자, 글자, 참/거짓 등 다양한 종류가 있어!

자료형은 데이터의 종류를 말해요.
자바스크립트가 데이터를 어떻게 처리할지 결정하는 중요한 기준이 됩니다.

1. 숫자형 (Number)

나이, 키, 점수처럼 계산할 수 있는 숫자예요.

JavaScript 
let age = 10;
let height = 130.5;
console.log(age + 5); // 덧셈 가능
Result

15

🔢
비유: 수학 시간에 쓰는 숫자 카드 상자예요.

2. 문자열 (String)

이름, 메시지 같은 글자예요. 반드시 따옴표("" 또는 '')로 감싸야 해요.

JavaScript 
let name = "코딩몬스터";
let hi = '안녕!';
console.log(hi + " " + name);
Result

"안녕! 코딩몬스터"

📝
비유: 문장을 만드는 글자 스티커 상자예요.

3. 불리언 (Boolean)

참(true) 또는 거짓(false) 딱 두 가지 값만 있어요.

JavaScript 
let isStudent = true; // 학생이다
let isAdult = false;  // 어른이 아니다
console.log(isStudent);
Result
ON (true)
💡
비유: 켜거나 끄는 스위치(ON/OFF)와 같아요.
자료형 총정리
자료형 설명 예시
Number 숫자 (계산 가능) 10, 3.14
String 글자 (따옴표 필수) "안녕", 'JS'
Boolean 참/거짓 (논리) true, false
