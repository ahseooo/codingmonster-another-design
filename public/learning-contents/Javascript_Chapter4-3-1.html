🌳 DOM (Document Object Model)

웹페이지는 사실 거대한 나무(Tree)처럼 생겼어! 🌲
자바스크립트로 이 나무의 가지를 흔들고 잎을 바꿔보자.

DOM은 HTML 문서를 객체(Object)로 구조화한 모델이에요.
자바스크립트는 이 DOM을 통해 HTML 태그를 찾고, 내용을 바꾸고, 스타일을 꾸밀 수 있답니다.

1. DOM 트리 구조

HTML 태그들은 부모와 자식 관계로 연결되어 있어요.

document
│
html
│
body
h1
p
🌳
비유: 뿌리(Root)에서 시작해 가지가 뻗어나가는 나무와 같아요.
각각의 태그(h1, p 등)는 나무의 잎사귀(Node)랍니다.

2. 요소 찾고 바꾸기

자바스크립트로 HTML 요소를 콕 집어서 내용을 바꿔볼까요?

JavaScript 
// 1. h1 태그를 찾는다
let title = document.querySelector("h1");

// 2. 내용을 바꾼다
title.textContent = "반가워요! JS";

// 3. 색깔을 바꾼다
title.style.color = "blue";
Preview

반가워요! JS

3. 자주 쓰는 DOM 도구들

이 도구들만 알면 웹페이지를 마음대로 조종할 수 있어요!

🔍

querySelector()

원하는 태그 하나를 찾아요.

document.querySelector("p")
✏️

textContent

태그 안의 글자를 바꿔요.

el.textContent = "안녕"
🎨

style

CSS 디자인을 바꿔요.

el.style.color = "red"
➕

createElement()

새로운 태그를 만들어요.

document.createElement("div")

4. 직접 해보기! 🎮

이 글자가 바뀔 거예요!

btn.addEventListener("click", function() { ... })
정리하기
  • DOM: HTML을 자바스크립트가 이해할 수 있게 만든 객체 모델.
  • document: 웹페이지 전체를 담당하는 가장 큰 객체.
  • 조작 순서: 요소 찾기(querySelector) → 바꾸기(textContent, style).
