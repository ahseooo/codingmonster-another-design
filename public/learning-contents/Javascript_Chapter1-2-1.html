💡 자바스크립트의 특징

자바스크립트는 다른 언어랑 뭐가 다를까? 🤔
바로 실행되고, 자유롭고, 어디서든 쓰이는 매력이 있어!

자바스크립트가 가진 특별한 성격들을 알아볼까요?
이 특징들 덕분에 웹 개발이 훨씬 재미있고 강력해진답니다.

🚀

1. 바로 실행돼요! (인터프리터)

코드를 작성하면 컴퓨터가 한 줄씩 바로 읽고 실행해요.
따로 '번역(컴파일)'하는 시간이 필요 없어서 결과를 빨리 볼 수 있죠.

🎭

2. 변수는 자유로워요! (동적 타입)

숫자였다가 글자가 될 수도 있어요!
자바스크립트는 "아~ 이건 숫자구나!" 하고 스스로 알아챈답니다.

💻

3. 브라우저에서 돌아요!

서버까지 갔다 올 필요 없이,
여러분의 컴퓨터(브라우저) 안에서 바로바로 실행돼요.

📦

4. 모든 게 '객체'예요!

사람, 자동차, 버튼... 모든 것을 객체(Object)로 다뤄요.
객체는 여러 정보를 담는 보물상자 같아요.

🖱️

5. 클릭하면 움직여요!

사용자가 클릭하거나 입력하는 순간을 감지해요.
"앗! 마우스가 움직였네?" 하고 즉시 반응하죠.

🎨

6. 자유로운 코딩 스타일

순서대로 짜도 되고, 함수로 묶어도 되고!
여러 가지 방식으로 유연하게 코드를 작성할 수 있어요.
