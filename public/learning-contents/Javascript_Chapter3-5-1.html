📚 문자열과 배열 함수 총정리

글자나 배열을 내 마음대로 자르고, 바꾸고, 합치는 마법 도구들을 소개할게! ✨

자바스크립트에는 문자열(String)과 배열(Array)을 다루는 유용한 내장 함수들이 아주 많아요.
자주 쓰이는 것들을 하나씩 알아볼까요?

1. 문자열 함수 (String Methods)

📝 글자 다루기
length
글자 수 세기
"Apple".length
➡ 5
toUpperCase()
대문자로 변환
"hi".toUpperCase()
➡ "HI"
toLowerCase()
소문자로 변환
"HI".toLowerCase()
➡ "hi"
slice(start, end)
글자 자르기
"Hello".slice(0,2)
➡ "He"
replace(old, new)
글자 바꾸기
"Hi JS".replace("JS","World")
➡ "Hi World"
split(separator)
문자열 ➡ 배열
"A,B".split(",")
➡ ["A", "B"]

2. 배열 함수 (Array Methods)

📦 상자 다루기
push(item)
맨 뒤에 추가 ➕
arr.push("C")
➡ ["A", "B", "C"]
pop()
맨 뒤 제거 ➖
arr.pop()
➡ ["A"]
unshift(item)
맨 앞에 추가 ➕
arr.unshift("Z")
➡ ["Z", "A", "B"]
shift()
맨 앞 제거 ➖
arr.shift()
➡ ["B"]
join(separator)
배열 ➡ 문자열
["A","B"].join("-")
➡ "A-B"
forEach(function)
하나씩 꺼내서 실행 (반복)
arr.forEach(item => console.log(item));
➡ A, B (차례대로 출력)
기억해두면 좋은 짝꿍!
↔️
split vs join
문자열을 쪼개면(split) 배열이 되고,
배열을 합치면(join) 문자열이 돼요!
➕➖
push/pop vs unshift/shift
뒤에서 넣고 빼기(push/pop)가
앞에서 하기(unshift/shift)보다 빨라요!
