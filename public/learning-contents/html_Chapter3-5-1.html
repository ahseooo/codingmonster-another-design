📂 파일 경로 이해하기 (Path)

이미지가 안 나오고 엑박(X)이 뜬다면?
99%는 파일 경로(주소)를 잘못 적어서 그래! 📍

HTML 파일이 "사진이나 다른 파일이 어디 있는지" 찾을 수 있도록 주소를 적는 방법이에요.
크게 상대 경로와 절대 경로로 나뉩니다.

1. 상대 경로 (Relative Path)

"현재 내 위치(HTML 파일)"를 기준으로 파일이 어디 있는지 찾아가는 방법이에요.

Case 1 같은 폴더에 있을 때
프로젝트_폴더
index.html 나(Here)
cat.jpg
src="cat.jpg"

가장 쉬워요! 그냥 파일 이름만 적으면 됩니다.

Case 2 하위(아래) 폴더에 있을 때
프로젝트_폴더
index.html 나(Here)
images (폴더)
cat.jpg
src="images/cat.jpg"

폴더이름/파일이름 순서로 적어요. (/는 '안에 있다'는 뜻)

Case 3 상위(위) 폴더에 있을 때
프로젝트_폴더
cat.jpg
pages (폴더)
index.html 나(Here)
src="../cat.jpg"

../는 "상위 폴더로 나가라!"는 명령이에요.

2. 절대 경로 (Absolute Path)

내 위치와 상관없이 고유한 주소(URL)를 사용하는 방법이에요.

HTML Code 
<img src="https://site.com/logo.png">
설명

http:// 또는 https://로 시작하는 인터넷 주소를 그대로 복사해서 넣으면 됩니다.

⚠️ 단, 원본 사이트에서 이미지를 삭제하면 우리 사이트에서도 안 보여요!

언제 무엇을 쓸까요?
  • 🏠
    상대 경로 (추천!)
    내 컴퓨터에서 작업할 때, 나중에 서버에 올릴 때도 경로가 유지되므로 가장 많이 써요.
  • 🌏
    절대 경로
    다른 웹사이트(구글, 네이버 등)에 있는 이미지를 가져올 때만 주로 사용해요.
