⌨️ 입력 요소 (Input Types)

<input> 태그는 변신의 귀재야! 🦸‍♂️
type 속성만 바꾸면 글자 창, 색상 판, 버튼으로 변신해!

사용자에게 정보를 입력받는 가장 중요한 태그인 <input>의
다양한 타입(type)들을 직접 체험해봅시다.

1. 기본 구조

닫는 태그가 없으며, type 속성이 가장 중요합니다.

<input type="...">

2. 자주 쓰는 타입 알아보기

type="text"

가장 기본적인 글자 입력 창

<input type="text">
type="color"

색상을 선택하는 팔레트

#667eea
<input type="color">
type="button"

클릭할 수 있는 버튼

<input type="button" value="클릭">
type="radio"

하나만 선택 (택1)

<input type="radio" name="...">

3. 여러 줄 입력 (<textarea>)

<input>은 한 줄만 가능하지만, 이 태그는 긴 글을 쓸 수 있어요.

HTML Code 
<textarea rows="4" placeholder="내용..."></textarea>
Result

배운 것들을 모아서 설문조사 폼을 완성해봅시다.

색상을 골라주세요
입력 타입 총정리
타입(type) 설명 예시

password 비밀번호 (가려짐) 비번 입력
