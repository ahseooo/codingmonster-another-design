🎨 글자 예쁘게 꾸미기

글씨가 너무 밋밋하지? 😐
굵게 만들거나 색깔을 바꿔서 강조해보자!

중요한 내용은 굵게 표시하고, 강조하고 싶은 부분은 색을 바꿔요.
밑줄을 그어서 눈에 띄게 할 수도 있답니다.

1. 굵게 만들기 (Bold)

<b> 또는 <strong> 태그를 사용해요.

HTML Code 
<p>
  나는 <b>굵은 글씨</b>를 좋아해요!
</p>
<p>
  이건 <strong>정말 중요한</strong> 내용이에요!
</p>
Browser Result

나는 굵은 글씨를 좋아해요!

이건 정말 중요한 내용이에요!

<b>

단순히 보기에만 굵어요.

VS
<strong>

컴퓨터에게 "이건 중요해!"라고 알려줘요.

2. 밑줄 긋기 (Underline)

<u> 태그로 글자 아래에 선을 그을 수 있어요.

HTML Code 
<p>
  중요한 부분에 <u>밑줄 쫙!</u> 그어보세요.
</p>
Browser Result

중요한 부분에 밑줄 쫙! 그어보세요.

3. 글자 색 바꾸기 (Color)

style="color: 색상이름;" 속성을 사용해요.

HTML Code 
<p style="color:red;">빨간색 사과</p>
<p style="color:blue;">파란색 하늘</p>
<p style="color:#4caf50;">초록색 나뭇잎</p>
Browser Result
