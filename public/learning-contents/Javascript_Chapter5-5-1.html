⏳ 시간 다루기 (setTimeout, setInterval)

"3초 뒤에 알려줘!" 또는 "1초마다 계속 실행해!" ⏰
자바스크립트로 시간을 지배하는 방법을 배워보자!

코드를 바로 실행하지 않고 기다렸다가 실행하거나,
일정한 간격으로 반복해서 실행할 때 사용하는 함수들이에요.

1. setTimeout (한 번만 기다리기)

정해진 시간이 지나면 딱 한 번 실행해요.

JavaScript
setTimeout(function() {
  console.log("3초 지났어요! ⏰");
}, 3000); // 3000ms = 3초
Demo
대기 중...
⏲️
비유: 전자레인지 타이머와 같아요.
"3분 뒤에 땡!" 하고 한 번 울리고 끝나죠?

2. setInterval (계속 반복하기)

정해진 시간 간격마다 계속해서 실행해요.

JavaScript
let count = 0;

setInterval(function() {
  count++;
  console.log(count + "초 경과");
}, 1000); // 1초마다 실행
Demo

💓
비유: 심장 박동이나 시계 초침과 같아요.
"똑-딱-똑-딱" 일정하게 계속 반복되죠!

3. 타이머 멈추기 (clear)

실행 중인 타이머를 취소할 수 있어요.

clearTimeout()

setTimeout 취소

clearTimeout(timerId);

clearInterval()

setInterval 취소

clearInterval(timerId);
차이점 정리
함수 반복 횟수 비유
setTimeout 1번 (지연 실행) ⏲️ 오븐 타이머
setInterval 무한 반복 (주기적 실행) 🕰️ 시계 초침
