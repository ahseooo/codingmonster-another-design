🔎 자바 개발 도구(JDK)가 뭐야?

요리할 땐 냄비가 필요하듯, 코딩에도 준비물이 있어! 자바로 코딩을 하려면 먼저 도구가 필요해!

JDK는 Java Development Kit의 줄임말이야.
자바 코드를 만들고 실행할 수 있게 도와주는 필수 도구지.

JDK 안에는 세 가지 중요한 친구들이 있어.

📦

JDK (자바 개발 도구)

자바로 코드를 만들고 실행할 수 있게 해줘.

💡 ‘게임도 만들고 플레이도 할 수 있는 게임 상자’
▶️

JRE (런타임 환경)

만든 자바 프로그램을 실제로 ‘실행’하게 도와줘.

💡 ‘게임을 플레이하기 위한 게임기’
⚙️

JVM (자바 가상 머신)

우리가 쓴 자바 코드를 컴퓨터가 알아듣게 번역해줘.

💡 ‘게임 CD를 읽는 게임기 안의 칩셋’

✔ 설치 전 준비

✔ 인터넷 연결 확인
✔ 컴퓨터 운영체제 확인 (Windows / Mac / Linux)
✔ 디스크 저장 공간 남겨두기
