HTML은 혼자서 일하지 않아요.
친구들과 함께 어울리며, 정해진 규칙대로 화면을 보여준답니다.

HTML의 6가지 특징
  • 1. 웹사이트의 뼈대
    제목, 글, 사진이 어디에 위치할지 구조를 잡아요.
  • 2. 태그(Tag)로 명령
    <h1>, <img> 처럼 괄호 안에 명령어를 담아 대화해요.
  • 3. 위에서 아래로 (순서)
    우리가 글을 쓰는 순서대로 브라우저가 하나씩 보여줘요.
  • 4. 짝꿍: CSS (꾸미기)
    HTML이 뼈대라면, CSS는 옷을 입히고 예쁘게 꾸며줘요.
  • 5. 짝꿍: JavaScript (동작)
    움직이거나 반응하는 기능은 자바스크립트가 도와줘요.
  • 6. 정적인 언어
    HTML은 눈에 보이는 것만 만들어요. (움직임은 친구 담당!)
