🔁 반복문 (Loop)

같은 일을 100번 반복해야 한다면? 😱
반복문을 쓰면 단 몇 줄로 끝낼 수 있어!

반복문은 코드를 원하는 횟수만큼 또는 조건이 맞을 때까지
계속해서 실행시켜주는 문법이에요.

1. 횟수가 정해져 있을 때 (for)

"사탕 5개 포장해!" 처럼 횟수가 명확할 때 사용해요.

JavaScript 
for (let i = 1; i <= 5; i++) {
  console.log("사탕 포장 " + i + "개 🍬");
}
Result
  • 사탕 포장 1개 🍬
  • 사탕 포장 2개 🍬
  • 사탕 포장 3개 🍬
  • 사탕 포장 4개 🍬
  • 사탕 포장 5개 🍬

2. 조건이 중요할 때 (while)

"사탕이 다 떨어질 때까지" 처럼 조건이 참인 동안 계속 반복해요.

JavaScript 
let candy = 1;

while (candy <= 3) {
  console.log("먹은 사탕 " + candy + "개 🍭");
  candy++; // 잊지 말고 증가시켜야 해요!
}
Result
  • 먹은 사탕 1개 🍭
  • 먹은 사탕 2개 🍭
  • 먹은 사탕 3개 🍭
