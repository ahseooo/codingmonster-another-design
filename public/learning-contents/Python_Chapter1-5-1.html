자주 발생하는 에러와 해결방법

프로그래밍을 하다 보면 에러는 자연스러운 일이니 너무 걱정하지 마!
대부분의 에러는 간단하게 해결할 수 있어. 가장 자주 발생하는 에러와 해결 방법을 알아볼까?

📁 들여쓰기 에러

들여쓰기가 잘못 사용되었을 때 발생하는 에러야!

잘못된 예
if x > 10:
print("큰 수입니다")  # 들여쓰기 에러 발생
올바른 예
if x > 10:
    print("큰 수입니다")
파이썬(Python)은 들여쓰기로 블록을 구분해! 들여쓰기를 제대로 하지 않으면 IndentationError 또는 unexpected indent 에러가 발생해!

if, for, while, def 뒤에 오는 줄은 스페이스 4칸 또는 Tab 1번 들여쓰기 해줘야 해!
🔤 따옴표 / 괄호 누락

따옴표나 괄호가 빠졌을 때 발생해!

잘못된 예
print("Hello)      # 닫는 따옴표 누락
print("Python!"    # 닫는 괄호 누락
올바른 예
print("Hello")
print("Python!")
따옴표나 괄호가 잘 열리거나 닫히지 않으면 SyntaxError가 발생해!

문자열은 "…", '…' 처럼 짝을 맞춰서 써야 해! 괄호 ( 를 열었으면 반드시 ) 로 닫아줘야 해!
🔠 변수 이름 오타

변수 이름을 잘못 썼을 때 발생하는 에러야!

잘못된 예
age = 10
print(agge)   # 변수 이름 오타
올바른 예
age = 10
print(age)
변수 이름이 다르면 NameError: name 'agge' is not defined 오류가 발생해!

변수를 사용할 때는 처음 만든 이름과 똑같이 써야 해! 파이썬은 대소문자를 구분하니까 Age, AGE, age는 전부 다른 변수야!
🔧 자료형 혼용 에러

숫자와 문자열을 잘못 섞어서 사용할 때 발생해!

잘못된 예
age = 10
text = "살입니다"
print(age + text)
올바른 예
age = 10
text = "살입니다"
print(str(age) + text)   # 10살입니다

# 또는
print(age, text)         # 10 살입니다
숫자(int)와 문자열(str)을 그대로 더하려고 하면 TypeError: can only concatenate str (not "int") to str 에러가 발생해!

숫자는 str()로 문자열로 변환한 뒤 사용해야 해!

에러 해결 꿀팁!

  • 에러 메시지를 무시하면 안돼! 가장 중요한 힌트야.
  • 에러가 표시된 줄 번호부터 확인하기.
  • Ctrl + S 로 자주 저장하기.
  • 문제가 해결되지 않으면 구글 검색으로 빠르게 해결!
