⚡ 이벤트 처리 (Event Handling)

클릭! 🖱️ 키보드 타닥! ⌨️
사용자가 무언가 할 때, "짠!" 하고 반응하게 만들어볼까?

이벤트(Event)는 웹페이지에서 일어나는 모든 사건을 말해요.
이벤트가 발생했을 때 실행할 코드를 연결하는 것을 이벤트 처리라고 합니다.

1. 자주 쓰는 이벤트 종류

웹페이지에서 어떤 일들이 일어날 수 있을까요?

🖱️

click

딸깍! 클릭했을 때

👆

mouseover

마우스를 올렸을 때

⌨️

keydown

키보드를 눌렀을 때

📝

input

글자를 입력할 때

2. 이벤트 연결하기 (addEventListener)

요소에게 "이런 일이 생기면, 저 일을 해!" 라고 귀띔해주는 거예요.

JavaScript 
let btn = document.querySelector("#myBtn");

// 클릭하면 알림창 띄우기
btn.addEventListener("click", function() {
  alert("버튼 클릭됨! 🎉");
});
Action
👂
비유: addEventListener는 "귀를 기울이고 있어라!"는 명령이에요.
"누가 클릭하면(click), 알려줘!" 라고 시키는 거죠.

3. 실전: 마우스에 반응하는 상자 📦

마우스를 올리면 색이 바뀌고, 떼면 원래대로 돌아오게 해볼까요?

JavaScript 
let box = document.querySelector("#box");

box.addEventListener("mouseover", function() {
  box.style.background = "orange";
  box.textContent = "올라왔다! 😮";
});

box.addEventListener("mouseout", function() {
  box.style.background = "lightblue";
  box.textContent = "나갔다... 😐";
});
Interactive Demo
마우스를 올려봐! 👆
핵심 문법
요소.addEventListener("이벤트이름", 함수);
  • 요소: 이벤트를 받을 대상 (버튼, 박스 등)
  • 이벤트이름: click, mouseover, keydown 등
  • 함수: 이벤트가 생기면 실행할 코드
