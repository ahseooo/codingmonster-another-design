글자에 색을 입히고,
이미지로 생동감을!

텍스트를 예쁘게 꾸미고, 링크와 이미지를 추가하여
더 풍성하고 연결된 웹 페이지를 만들어보세요!

학습 목차

  • 글자 꾸미기 (굵게, 밑줄, 색상)
  • 하이퍼링크 (a 태그, 새 탭 열기)
  • 이미지 넣기 (img 태그, 속성)
  • 목록 만들기 (순서 있는/없는 목록)
  • 파일 경로 이해하기 (상대/절대 경로)

학습 목표

  • 텍스트 스타일을 변경하여 내용을 강조할 수 있다.
  • 다른 페이지나 사이트로 이동하는 링크를 만들 수 있다.
  • 웹 페이지에 이미지를 올바르게 삽입할 수 있다.
  • 정보를 목록으로 정리하고, 올바른 파일 경로를 지정할 수 있다.

학습 진행 현황

5
학습 주제
~2.5h
