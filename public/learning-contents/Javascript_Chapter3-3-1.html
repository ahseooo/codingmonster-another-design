🗂️ 배열과 객체 함께 쓰기 (JSON)

쇼핑몰의 상품 목록은 어떻게 저장할까? 🛒
배열 안에 객체를 넣어서 많은 정보를 한 번에 관리할 수 있어!

배열(리스트)과 객체(정보 묶음)를 함께 사용하면,
복잡하고 많은 양의 데이터를 체계적으로 관리할 수 있어요. 이것이 바로 JSON 형식의 기초랍니다.

1. 배열 속에 객체 넣기

여러 사람의 정보를 관리한다고 생각해보세요. 배열 안에 객체를 쏙쏙! 넣으면 됩니다.

JavaScript 
let students = [
  { name: "지우", age: 10 },
  { name: "이슬", age: 12 },
  { name: "웅이", age: 15 }
];

console.log(students[0]); // 첫 번째 학생
console.log(students[1].name); // 두 번째 학생 이름
Students List
👦
지우 (10세)
👧
이슬 (12세)
🧑
웅이 (15세)
📚
비유: 출석부(배열) 안에 학생들의 상세 정보가 적힌 카드(객체)가 순서대로 꽂혀 있는 것과 같아요.

2. 객체 속에 배열 넣기

한 사람(객체)이 좋아하는 음식(배열)을 여러 개 가질 수도 있겠죠?

JavaScript 
let pikachu = {
  name: "피카츄",
  type: "전기",
  skills: ["백만볼트", "전광석화"]
};

console.log(pikachu.skills[0]); // "백만볼트"
Pikachu Info

⚡ 피카츄

타입: 전기

백만볼트 전광석화
🎒
비유: 가방(객체) 안에 필통(배열)이 있고, 그 필통 안에 연필들(데이터)이 들어있는 구조예요.
데이터 구조 정리
구조 설명 접근 방법
배열 속 객체 목록 안에 상세 정보 list[0].name
객체 속 배열 정보 안에 목록 info.items[0]
