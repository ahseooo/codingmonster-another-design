📬 매개변수와 반환 (Parameter & Return)

로봇에게 재료를 주면 완성품을 만들어줘! 🤖
함수랑 데이터를 주고받는 방법을 배워보자.

함수는 단순히 명령만 실행하는 게 아니라,
외부에서 값을 받아(매개변수) 처리하고, 그 결과를 돌려줄(반환) 수도 있어요.

1. 재료 주기 (매개변수)

함수를 실행할 때 필요한 값을 전달해요.

JavaScript 
function makeJuice(fruit) {
  console.log(fruit + " 주스 완성! 🍹");
}

makeJuice("사과");
makeJuice("딸기");
Mixer Result
🍎 → 사과 주스 완성! 🍹
🍓 → 딸기 주스 완성! 🍹
🍎
비유: 믹서기(함수)에 과일(매개변수)을 넣는 것과 같아요.
무엇을 넣느냐에 따라 결과가 달라지죠!

2. 결과 받기 (return)

함수가 계산한 최종 결과값을 밖으로 꺼내요.

JavaScript 
function add(a, b) {
  return a + b;
}

let result = add(3, 4);
console.log(result); // 7
Calculation
3 + 4 ➞ 7
📤
비유: 자판기에 동전을 넣으면(매개변수), 음료수(반환값)가 툭! 하고 나오는 것과 같아요.
용어 정리
용어 설명 예시
매개변수
(Parameter)		 함수가 받는 변수 이름 (fruit)
인자
(Argument)		 실제로 넣어주는 값 "사과"
반환
(Return)		 함수가 돌려주는 결과 return a + b;
