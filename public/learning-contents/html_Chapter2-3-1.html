✂️ 줄바꿈, 수평선, 그리고 주석

글을 쓰다 보면 줄을 바꾸거나 선을 긋고 싶지?
그리고 나만 볼 수 있는 비밀 메모도 남길 수 있어! 🤫

화면을 깔끔하게 정리하는 태그들과,
개발자를 위한 주석(Comment) 기능을 배워봐요.

1. 줄바꿈 (Line Break)

<br>은 엔터키처럼 줄을 바꿔주는 역할을 해요. 닫는 태그가 없어요!

HTML Code 
<p>
  안녕!<br>
  나는 HTML이야.<br>
  줄이 바뀌었지?
</p>
Browser Result

안녕!
나는 HTML이야.
줄이 바뀌었지?

2. 수평선 (Horizontal Rule)

<hr>은 내용을 구분하기 위해 가로선을 그어요. 이것도 닫는 태그가 없어요.

HTML Code 
<h3>오늘의 메뉴</h3>
<hr>
<p>떡볶이</p>
<p>김밥</p>
Browser Result

오늘의 메뉴

떡볶이

김밥

3. 주석 (Comment)

<!-- --> 안에 쓴 내용은 화면에 보이지 않아요. 개발자만의 메모장이에요!

HTML Code 
<!-- 여기는 비밀 메모예요 -->
<h3>HTML 배우기</h3>
<!-- 나중에 내용을 추가해야지 -->
<p>화면에 보이는 내용</p>
Browser Result

HTML 배우기

화면에 보이는 내용

👻 (주석은 안 보여요!)

닫는 태그가 없는 친구들?
  • ↩️
    <br> (줄바꿈)
    그냥 <br>만 쓰면 끝! (내용을 감싸지 않으니까요)
  • ➖
    <hr> (수평선)
    이것도 선만 그으면 되니까 닫을 필요가 없어요.
