✨ 특수문자, 들여쓰기, 정렬

화면에 < 같은 기호를 어떻게 쓸까?
그리고 글씨를 가운데로 보내려면? 🤔

특수 기호를 표시하는 방법과, 코드를 깔끔하게 정리하는 들여쓰기,
그리고 텍스트를 원하는 위치로 정렬하는 법을 배워봐요.

1. 특수문자 (Special Characters)

HTML에서 <나 >를 그냥 쓰면 태그로 오해해요. 그래서 특수 코드를 써야 합니다.

기호 HTML 코드 (이렇게 써야 해요!) 설명
< &lt; 작다 (Less Than)
> &gt; 크다 (Greater Than)
공백 &nbsp; 띄어쓰기 한 칸
" &quot; 큰따옴표
♥ &hearts; 하트 모양

2. 들여쓰기 (Indentation)

들여쓰기는 화면엔 안 보이지만, 코드를 읽기 쉽게 만들어줘요. 부모 태그 안쪽은 항상 들여쓰세요!

😵 보기 힘든 코드 
<html>
<body>
<h1>안녕</h1>
<p>복잡해 보여요</p>
</body>
</html>
😊 잘 정리된 코드 
<html>
  <body>
    <h1>안녕</h1>
    <p>훨씬 보기 좋죠?</p>
  </body>
</html>

3. 텍스트 정렬 (Text Align)

style="text-align: 방향;"을 사용해서 글자를 정렬할 수 있어요.

HTML Code 
<p style="text-align:left;">
  왼쪽 정렬 (기본값)
</p>

<p style="text-align:center;">
  가운데 정렬
</p>

<p style="text-align:right;">
  오른쪽 정렬
</p>
