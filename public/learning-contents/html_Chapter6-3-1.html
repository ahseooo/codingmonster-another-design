📝 미니 프로젝트: 설문조사 만들기

친구들의 취향을 조사해볼까? 🔍
이름, 나이, 좋아하는 것을 물어보는 설문조사 페이지를 만들어보자!

<form> 태그와 다양한 입력 요소들을 사용해서
사용자에게 정보를 입력받는 양식을 만들어봅시다.

📋 오늘의 미션

1
제목과 이름 입력
설문 제목(h1)과 이름 입력칸(input type="text")을 만드세요.
2
학년 입력 (숫자)
학년은 숫자만 들어가야 해요. type="number"를 써보세요.
3
좋아하는 과목 (다중 선택)
여러 개 고를 수 있는 checkbox를 사용하세요. (국어, 수학, 체육 등)
4
점심 메뉴 (단일 선택)
하나만 고를 수 있는 radio 버튼을 만드세요. (급식, 도시락)
5
제출 버튼
마지막에 button을 만들어 설문을 완성하세요!
