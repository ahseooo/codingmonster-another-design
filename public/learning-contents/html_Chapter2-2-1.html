📝 제목(Heading)과 문단(Paragraph)

책에도 큰 제목, 작은 제목, 그리고 내용이 있지?
웹페이지도 똑같아! h태그와 p태그를 써보자.

글씨를 크고 굵게 만들고 싶을 때는 제목 태그를,
긴 줄글을 쓸 때는 문단 태그를 사용해요.

1. 제목 태그 (Heading)

<h1>부터 <h6>까지 숫자가 커질수록 글씨는 작아져요.

HTML Code 
<h1>가장 큰 제목 (h1)</h1>
<h2>두 번째 큰 제목 (h2)</h2>
<h3>세 번째 제목 (h3)</h3>
<h4>네 번째 제목 (h4)</h4>
<h5>다섯 번째 제목 (h5)</h5>
<h6>가장 작은 제목 (h6)</h6>
Browser Result

가장 큰 제목 (h1)

두 번째 큰 제목 (h2)

세 번째 제목 (h3)

네 번째 제목 (h4)

다섯 번째 제목 (h5)
가장 작은 제목 (h6)

2. 문단 태그 (Paragraph)

글의 본문을 쓸 때는 <p> 태그를 사용해요. 자동으로 줄이 바뀝니다.

HTML Code 
<p>
  안녕하세요! 저는 웹페이지를 배우고 있어요.
</p>
<p>
  p태그를 쓰면 글이 한 덩어리가 돼요.
  자동으로 줄바꿈도 된답니다!
</p>
Browser Result

안녕하세요! 저는 웹페이지를 배우고 있어요.

p태그를 쓰면 글이 한 덩어리가 돼요. 자동으로 줄바꿈도 된답니다!

책으로 비유하자면?
  • 📑
    <h1> ~ <h6>
    책의 장(Chapter) 제목이나 소제목이에요. (h1이 제일 중요!)
  • 📝
    <p>
    책 안에 있는 일반적인 글 내용(본문)이에요.
