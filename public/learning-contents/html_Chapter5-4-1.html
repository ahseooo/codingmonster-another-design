웹페이지가 너무 조용하다면? 🎵
유튜브 영상이나 배경음악을 넣어보자!

다른 웹페이지를 불러오거나(iframe),
오디오와 비디오 파일을 직접 재생하는 방법을 배워봅시다.

1. 다른 페이지 불러오기 (<iframe>)

내 페이지 안에 작은 창(Frame)을 뚫어서 다른 사이트나 지도를 보여줘요.

HTML Code 
<iframe src="https://youtube.com/..."
  width="100%" height="200">
</iframe>
YouTube Embed Example
▶

유튜브 영상이 여기에 나옵니다

2. 소리 넣기 (<audio>)

음악이나 효과음을 재생해요. controls 속성이 있어야 재생 버튼이 보입니다.

HTML Code 
<audio controls>
  <source src="music.mp3">
</audio>
Audio Player
0:00 / 3:45

3. 영상 넣기 (<video>)

동영상을 직접 재생해요. width로 크기를 조절하는 게 좋아요.

HTML Code 
<video controls width="100%">
  <source src="movie.mp4">
</video>
Video Player
▶
핵심 속성 정리
속성 설명 사용 태그
src 파일 경로/주소 iframe, audio, video
controls 재생 버튼 표시 audio, video
autoplay 자동 재생 audio, video
loop 무한 반복 audio, video
