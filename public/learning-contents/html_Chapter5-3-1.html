1. 웹페이지의 기본 3단 구조

대부분의 웹사이트는 이 3가지 영역으로 나뉩니다.

<header> 헤더 (머리말)
<main> 메인 (본문 내용)
<footer> 푸터 (바닥글)

2. 코드로 구현하기

HTML 코드만으로 구조를 잡고, 간단한 CSS로 색상을 입혀봤어요.

HTML Code 
<!DOCTYPE html>
<html>
<head>
  <style>
    header { background: #8661FF; ... }
    main   { background: #FFFC94; ... }
    footer { background: #503E9A; ... }
  </style>
</head>
<body>

  <header>
    <h1>내 사이트</h1>
  </header>

  <main>
    <p>여기에 본문 내용이 들어갑니다.</p>
  </main>

  <footer>
    <p>저작권 정보</p>
  </footer>

</body>
</html>

3. 각 영역의 역할 복습

🏠

Header

  • 사이트 로고
  • 메뉴 (네비게이션)
  • 검색창
📄

Main

  • 실제 콘텐츠
  • 블로그 글, 뉴스
  • 상품 목록
🦶

Footer

  • 저작권 표시 (©)
  • 연락처, 주소
  • 이용약관 링크
💡
