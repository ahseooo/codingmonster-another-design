📦 변수 선언 (let, const, var)

데이터를 담아두는 상자(변수)를 만들어볼까?
상자마다 특징이 다르니까 상황에 맞게 골라 써야 해!

변수는 데이터를 저장하는 공간이에요. 자바스크립트에서는
var, let, const 세 가지 방법으로 변수를 만들 수 있습니다.

1. var (옛날 상자)

예전부터 쓰던 방식이지만, 지금은 잘 쓰지 않아요.

JavaScript 
var candy = "딸기맛";
var candy = "포도맛"; // 또 만들어도 됨 (문제!)
console.log(candy);
Console Result

"포도맛"

📦
비유하자면:
이름이 똑같은 상자를 여러 개 만들어도 되는 마법 상자!
하지만 나중에 헷갈려서 실수를 많이 해요. 😅

2. let (요즘 쓰는 상자)

가장 많이 쓰는 방식이에요. 내용은 바꿀 수 있지만, 이름은 중복해서 못 만들어요.

JavaScript 
let toy = "로봇";
toy = "인형"; // 내용 바꾸기 OK
// let toy = "자동차"; // 에러! 이름 중복 안됨
console.log(toy);
Console Result

"인형"

🎁
비유하자면:
이름표는 한 번만 붙일 수 있고, 안에 든 물건은 바꿔 넣을 수 있는 상자!

3. const (잠긴 상자)

Constant(상수)의 줄임말이에요. 한 번 정하면 절대 바꿀 수 없어요.

JavaScript 
const name = "홍길동";
// name = "임꺽정"; // 에러! 바꾸기 금지
console.log(name);
Console Result

"홍길동"

Uncaught TypeError: Assignment to constant variable.

🔒
비유하자면:
자물쇠로 꽉 잠가서 절대 내용을 바꿀 수 없는 상자!
한눈에 비교하기
종류 이름 다시 만들기 내용 바꾸기 추천
var ⭕ 가능 ⭕ 가능 비추천 👎
let ❌ 불가능 ⭕ 가능 추천 👍
const ❌ 불가능 ❌ 불가능 고정값 👍
