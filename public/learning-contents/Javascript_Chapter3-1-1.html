📦 함수 선언과 호출

똑같은 일을 매번 다시 적기 힘들지? 😫
명령을 모아놓은 상자(함수)를 만들어서 필요할 때마다 꺼내 쓰자!

함수는 특정 작업을 수행하는 코드 묶음이에요.
마치 "특별한 일을 하는 로봇"을 미리 만들어두고, 필요할 때마다 부르는 것과 같아요.

1. 로봇 만들기 (함수 선언)

"쿠키 굽는 로봇"을 만들어볼까요? 할 일을 순서대로 적어주면 돼요.

JavaScript 
function bakeCookie() {
  console.log("1. 밀가루 넣기");
  console.log("2. 설탕 넣기");
  console.log("3. 굽기 🔥");
}
Robot Status

쿠키 굽는 로봇 대기중...

🤖
비유: "이런 로봇이 있어!" 하고 설계도를 만든 상태예요.
아직 일은 안 해요. 부를 때까지 기다리고 있죠.

2. 로봇 부르기 (함수 호출)

만들어둔 로봇에게 "일해!" 하고 명령을 내려야 비로소 움직여요.

JavaScript 
bakeCookie(); // 로봇 작동!
bakeCookie(); // 한 번 더!
Result
  • 1. 밀가루 넣기
  • 2. 설탕 넣기
  • 3. 굽기 🔥
  • ---
  • 1. 밀가루 넣기
  • 2. 설탕 넣기
  • 3. 굽기 🔥
📢
비유: 로봇의 이름을 부르는 것(호출)이에요.
이름 뒤에 ()를 붙여야 실행된답니다!
