웹페이지를 살아 움직이게!
DOM과 이벤트 🖱️

객체의 행동을 정의하는 메서드를 이해하고,
DOM을 통해 HTML을 조작하여 사용자와 소통해보세요!

학습 목차

  • 객체 생성과 접근
  • 객체 메서드 (this 키워드)
  • DOM (Document Object Model)의 이해
  • HTML 요소 선택과 조작
  • 이벤트 처리 (버튼 클릭 등)

학습 목표

  • 객체 내부에 함수(메서드)를 만들어 동작을 정의할 수 있다.
  • DOM 트리 구조를 이해하고 원하는 요소를 선택할 수 있다.
  • 자바스크립트로 HTML 내용과 스타일을 동적으로 변경할 수 있다.
  • 사용자의 행동(이벤트)에 반응하는 웹페이지를 구현한다.
