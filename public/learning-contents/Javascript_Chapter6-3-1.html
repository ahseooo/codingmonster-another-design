🎰 미니 프로젝트: 랜덤 이름 추첨기

오늘 청소 당번은 누구? 점심 메뉴는 뭐 먹지? 🤔
결정하기 힘들 때 랜덤으로 뽑아주는 기계를 만들어보자!

배열에 후보들을 넣고, 버튼을 누르면 그중 하나를 무작위로 뽑아서 보여주는 프로그램을 만듭니다.

1. 오늘의 미션 📋

1

후보 준비: 이름 목록(배열) 만들기
(예: "철수", "영희", "민수")

2

화면 구성: "추첨하기" 버튼 만들기

3

랜덤 로직: 배열 길이(length) 안에서 랜덤 숫자 뽑기

4

결과 표시: 뽑힌 이름을 화면에 크게 보여주기

2. 코드 작성하기 ⌨️

왼쪽 에디터에 코드를 작성하고 실행해 보세요.

HTML + JS
결과 화면
핵심 코드

배열[Math.floor(Math.random() * 배열.length)]
이 공식 하나면 뭐든지 랜덤으로 뽑을 수 있어요!
