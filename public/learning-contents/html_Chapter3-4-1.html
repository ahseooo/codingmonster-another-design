📋 목록 만들기 (List)

쇼핑 리스트나 순서가 있는 할 일을 정리할 때,
목록(List) 태그를 사용하면 깔끔하게 정리할 수 있어!

목록은 크게 두 가지로 나뉘어요.
순서가 없는 목록 <ul>과 순서가 있는 목록 <ol>입니다.

1. 순서 없는 목록 (<ul>)

Unordered List의 약자예요. 앞에 점(Bullet)이 찍혀요.

HTML Code 
<ul>
  <li>사과</li>
  <li>바나나</li>
  <li>포도</li>
</ul>
Browser Result
  • 사과
  • 바나나
  • 포도

2. 순서 있는 목록 (<ol>)

Ordered List의 약자예요. 앞에 숫자(1, 2, 3...)가 붙어요.

HTML Code 
<ol>
  <li>양치하기</li>
  <li>세수하기</li>
  <li>아침 먹기</li>
</ol>
Browser Result
  1. 양치하기
  2. 세수하기
  3. 아침 먹기
꼭 기억하세요!
목록 안의 각 항목은 반드시 <li> (List Item) 태그로 감싸야 해요.
<ul>이나 <ol> 바로 아래에 글씨를 쓰면 안 돼요! 🙅‍♂️
무엇이 다른가요?
  • ●
    <ul> (Unordered)
    순서가 중요하지 않을 때 (예: 쇼핑 목록, 재료)
  • 1.
    <ol> (Ordered)
    순서가 중요할 때 (예: 요리 순서, 랭킹)
